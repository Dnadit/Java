# DataStructure(JAVA)

+ 실습에서 배열은 객체배열로 진행함.

+ ch5 : 재귀알고리즘 
  + 팩토리얼, 하노이 탑.

+ ch09 : 트리

  + OpenAddressing : int형을 속성으로 가지는 노드로 이루어진 버젼, 객체를 속성으로 가지는 노트로 이루어진 버젼 구현
